JavaScript DOM 조작법 정리

DOM

DOM(Document Object Model)이란?

DOM은 브라우저가 HTML 문서를 읽어서 만든 객체 구조입니다. HTML의 태그, 속성, 텍스트가 각각 객체처럼 표현되기 때문에 JavaScript로 화면의 글자, 스타일, 값, 요소를 읽거나 바꿀 수 있습니다.

Selector

Selector로 DOM 요소 가져오기

JavaScript는 CSS 선택자와 비슷한 방식으로 DOM 요소를 찾을 수 있습니다. 한 개의 요소가 필요하면 querySelector()를 사용하고, 여러 요소가 필요하면 querySelectorAll()을 사용합니다. 

const display = document.querySelector("[data-display]");
const buttons = document.querySelectorAll(".calc-key");

buttons.forEach((button) => {
    console.log(button.textContent);
});
  • document.querySelector("#title"): id가 title인 첫 번째 요소를 가져옵니다.
  • document.querySelector(".item"): class가 item인 첫 번째 요소를 가져옵니다.
  • document.querySelectorAll("button"): 모든 button 요소를 가져옵니다.

Event

Event란?

Event는 사용자의 행동이나 브라우저 상태 변화가 발생했다는 신호입니다. 클릭, 키보드 입력, 마우스 이동, 페이지 로딩, 폼 제출 같은 동작이 대표적인 이벤트입니다.

EventListener

EventListener란?

EventListener는 특정 이벤트가 발생했을 때 실행할 함수를 등록하는 방법입니다. 예를 들어 계산기 버튼에 클릭 이벤트 리스너를 등록하면 버튼을 누를 때마다 계산 로직을 실행할 수 있습니다.

Purpose

EventListener를 사용하는 목적과 이유

HTML 구조와 JavaScript 동작을 분리할 수 있고, 같은 요소에 여러 동작을 유연하게 연결할 수 있습니다. 사용자의 입력에 맞춰 화면을 즉시 바꾸는 인터랙티브 웹 페이지를 만들 때 꼭 필요합니다.

Define

EventListener 정의하기

addEventListener()의 첫 번째 인수에는 이벤트 이름을 넣고, 두 번째 인수에는 실행할 함수를 넣습니다. 

const clearButton = document.querySelector("[data-action='clear']");

clearButton.addEventListener("click", () => {
    console.log("계산기를 초기화합니다.");
});

Apply

EventListener 응용

여러 버튼에 같은 이벤트를 연결할 때는 querySelectorAll()과 forEach()를 함께 사용할 수 있습니다. 각 버튼의 data-action 값에 따라 다른 함수를 실행하면 계산기처럼 버튼이 많은 화면도 깔끔하게 관리할 수 있습니다. 

const buttons = document.querySelectorAll(".calc-key");

buttons.forEach((button) => {
    button.addEventListener("click", () => {
        const action = button.dataset.action;
        const value = button.dataset.value;

        handleAction(action, value);
    });
});

Web API

Web API의 정의

Web API는 브라우저가 JavaScript에 제공하는 기능 모음입니다. JavaScript 언어 자체에는 없는 화면 조작, 이벤트 처리, 시간 제어, 저장소 사용, 서버 통신 같은 기능을 브라우저의 API로 사용할 수 있습니다.

Examples

Web API의 예시

  • DOM API: querySelector(), createElement(), append()처럼 HTML 요소를 찾고 수정합니다.
  • Event API: addEventListener()로 클릭, 입력, 제출 같은 이벤트를 처리합니다.
  • Timer API: setTimeout(), setInterval()로 일정 시간 뒤 또는 반복 실행을 처리합니다.
  • Storage API: localStorage, sessionStorage로 브라우저에 간단한 데이터를 저장합니다.
  • Fetch API: fetch()로 서버에 데이터를 요청하거나 응답을 받아옵니다.
