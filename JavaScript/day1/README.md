# JavaScript Day1

웹 브라우저 Console에서 실행 가능한 사칙연산 계산기 과제입니다.

## 파일

- `index.html`: 콘솔 계산기 실행 페이지
- `calculator.js`: 계산기 로직
- `blog.html`: JavaScript 자료형, 변수 선언, 기본 문법 정리 글
- `style.css`: 계산기 페이지와 정리 글 공통 스타일

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. 개발자 도구의 Console 탭을 엽니다.
3. 아래처럼 계산식을 입력합니다.

```js
calculator("10 + 2 * 3 - 4 / 2");
calculate("(10 + 20) * 3 - 40 / 2");
```

## 구현 내용

- `+`, `-`, `*`, `/` 사칙연산 지원
- 긴 계산식 지원
- `*`, `/`가 `+`, `-`보다 먼저 계산되는 우선순위 적용
- 괄호 계산과 음수, 소수 계산 지원
