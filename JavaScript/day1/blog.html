JavaScript 자료형, 변수, 기본 문법 정리

JavaScript Day1 선택 과제를 블로그 글 형식으로 정리했습니다.

Type

JavaScript의 자료형

JavaScript는 값의 종류에 따라 자료형이 나뉩니다. 기본 자료형에는 숫자, 문자열, 불리언, null, undefined 등이 있습니다. 

let age = 20;              // number
let name = "김지현";       // string
let isStudent = true;      // boolean
let emptyValue = null;     // null
let notAssigned;           // undefined
  • number: 정수와 소수를 모두 표현합니다.
  • string: 문자 데이터를 표현합니다.
  • boolean: true 또는 false 값을 가집니다.
  • null: 값이 없다는 것을 개발자가 의도적으로 표현합니다.
  • undefined: 값이 아직 할당되지 않은 상태입니다.

Variable

변수 선언: let과 const

let은 값이 바뀔 수 있는 변수를 선언할 때 사용하고, const는 다시 할당하지 않을 값을 선언할 때 사용합니다. 

let score = 80;
score = 90;

const course = "JavaScript";
// course = "HTML"; 다시 할당할 수 없음

Operation

연산 방법

JavaScript에서는 산술 연산자를 사용해 숫자를 계산할 수 있습니다. Math 내장 객체를 사용하면 반올림, 최댓값, 랜덤값 같은 계산도 할 수 있습니다. 

10 + 3;          // 더하기
10 - 3;          // 빼기
10 * 3;          // 곱하기
10 / 3;          // 나누기
10 % 3;          // 나머지

Math.round(3.6); // 4
Math.max(1, 8);  // 8
Math.random();   // 0 이상 1 미만의 난수

Compare

비교 연산과 타입 확인

비교 연산자는 값의 크기나 일치 여부를 확인합니다. ==는 타입 변환 후 비교하고, ===는 값과 타입을 모두 비교하므로 보통 === 사용이 더 안전합니다. 

10 > 5;           // true
10 <= 10;         // true
"5" == 5;         // true
"5" === 5;        // false

typeof 10;        // "number"
typeof "hello";   // "string"

Logic

논리 연산

논리 연산자는 여러 조건을 조합할 때 사용합니다. 

const hasEmail = true;
const hasPassword = true;

hasEmail && hasPassword; // and: 둘 다 true일 때 true
hasEmail || hasPassword; // or: 하나라도 true면 true
!hasEmail;              // not: 값을 반대로 바꿈

String

문자열 연산

문자열은 +로 이어 붙일 수 있고, 템플릿 리터럴을 사용하면 변수와 문장을 더 읽기 쉽게 조합할 수 있습니다. 

const name = "지현";
const goal = "PM";

"안녕하세요, " + name + "입니다.";
`안녕하세요, ${name}입니다. 목표는 ${goal}입니다.`;

Python에서는 숫자와 문자열을 바로 더하면 오류가 나지만, JavaScript에서는 숫자가 문자열로 변환되어 이어 붙여질 수 있습니다. 예를 들어 10 + "20"은 "1020"이 됩니다.

Condition

조건문

조건문은 상황에 따라 다른 코드를 실행할 때 사용합니다. JavaScript에서는 elif가 아니라 else if를 사용합니다. 

const score = 85;

if (score >= 90) {
    console.log("A");
} else if (score >= 80) {
    console.log("B");
} else {
    console.log("C");
}

Loop

반복문

반복문은 같은 작업을 여러 번 실행할 때 사용합니다. 

for (let i = 1; i <= 3; i += 1) {
    console.log(i);
}

let count = 1;
while (count <= 3) {
    console.log(count);
    count += 1;
}

Array

배열

배열은 여러 값을 순서대로 저장하는 자료형입니다. 인덱스는 0부터 시작합니다. 

const skills = ["HTML", "CSS", "JavaScript"];

skills[0];              // "HTML"
skills.push("GitHub");  // 값 추가
skills.length;          // 배열 길이

for (const skill of skills) {
    console.log(skill);
}

Object

객체 자료형

객체는 이름이 있는 속성들을 묶어서 저장하는 자료형입니다. 

const user = {
    name: "김지현",
    job: "PM",
};

user.name;              // 속성 읽기
user.email = "mail@example.com"; // 속성 추가
user.job = "Product Manager";    // 속성 수정
delete user.email;      // 속성 제거
"name" in user;         // 속성 존재 확인

Function

함수

함수는 입력값을 받아 정해진 작업을 수행하고 결과를 반환할 수 있는 코드 묶음입니다. 함수는 함수명, 매개변수, 반환값으로 구성됩니다. 

function add(left, right) {
    return left + right;
}

const multiply = (left, right) => {
    return left * right;
};

const subtract = function (left, right) {
    return left - right;
};

add(1, 2);              // 기본 호출
const result = add(3, 4); // 값에 할당
[1, 2, 3].map((item) => item * 2); // 함수 전달

함수 내부에서 선언한 변수는 보통 함수 안에서만 사용할 수 있습니다. 이를 scope라고 합니다. 내부 scope는 바깥 scope의 변수를 찾을 수 있는데, 이것을 scope 연쇄라고 합니다. 안쪽 변수 이름이 바깥 변수 이름과 같으면 안쪽 변수가 우선 사용되며, 이를 변수 가리기라고 합니다.
