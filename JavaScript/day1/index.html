웹 브라우저 Console 계산기

브라우저 콘솔에서 사칙연산 계산식을 입력하면 연산자 우선순위에 맞춰 결과를 계산합니다.

Console

실행 방법

  1. 이 페이지를 브라우저에서 엽니다.
  2. 개발자 도구 Console 탭을 엽니다.
  3. calculator("10 + 2 * 3 - 4 / 2")를 입력합니다.

Function

사용 가능한 함수

  • calculator("계산식"): 결과를 콘솔에 출력하고 숫자를 반환합니다.
  • calculate("계산식"): 결과 숫자만 반환합니다.

Examples

테스트 계산식

calculator("1 + 2 * 3") calculator("100 / 5 + 8 * 2 - 3") calculator("10 + 20 * 3 - 40 / 2 + 7") calculator("(10 + 20) * 3 - 40 / 2") calculator("-10 + 3 * 8") calculator("3.5 * 2 + 1.25")

Required

필수 조건 반영

4칙연산 +, -, *, / 계산 지원
긴 계산식 여러 개의 숫자와 연산자가 이어진 계산식 처리
우선순위 *, /를 +, -보다 먼저 계산
